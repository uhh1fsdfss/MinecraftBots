# 🧠 ИИ Чат - Руководство

## 🌟 Обзор

ИИ чат система позволяет боту вести естественные разговоры с игроками, используя технологии OpenAI GPT. Бот понимает контекст игры, помнит предыдущие разговоры и отвечает как настоящий игровой персонаж.

## ⚙️ Настройка

### Получение OpenAI API ключа

1. **Регистрация**: Зайдите на [platform.openai.com](https://platform.openai.com)
2. **Создание ключа**: Перейдите в раздел [API Keys](https://platform.openai.com/api-keys)
3. **Новый ключ**: Нажмите "Create new secret key"
4. **Копирование**: Скопируйте ключ и сохраните в безопасном месте

### Конфигурация в .env файле

```env
# OpenAI Configuration
OPENAI_API_KEY=sk-your-api-key-here
OPENAI_MODEL=gpt-3.5-turbo

# Chat Settings
MAX_CHAT_HISTORY=50
RESPONSE_DELAY=1000
```

### Параметры настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `OPENAI_API_KEY` | API ключ OpenAI | - |
| `OPENAI_MODEL` | Модель GPT | gpt-3.5-turbo |
| `MAX_CHAT_HISTORY` | Максимум сообщений в памяти | 50 |
| `RESPONSE_DELAY` | Задержка ответа (мс) | 1000 |

## 🔧 Настройка поведения

### Частота ответов
Бот отвечает не на все сообщения, чтобы не спамить:
- **Прямое обращение**: всегда отвечает
- **Вопросы**: отвечает на вопросы
- **Упоминание**: реагирует на упоминание имени
- **Случайные сообщения**: отвечает в 20% случаев

### Настройка в коде
```javascript
// В src/ai/AIChat.js
const botMentioned = message.toLowerCase().includes(this.bot.username.toLowerCase()) ||
                   message.includes('бот') ||
                   message.includes('помощник') ||
                   message.endsWith('?');

if (!botMentioned) {
    // Изменить вероятность ответа
    if (Math.random() > 0.2) { // 20% шанс ответа
        return;
    }
}
```

## 🚫 Ограничения и решения

### Частые проблемы

**Бот не отвечает**
- Проверьте API ключ
- Проверьте баланс OpenAI
- Убедитесь что упоминаете бота

**Странные ответы**
- Проверьте системный промпт
- Очистите историю чата
- Перезапустите бота

**Медленные ответы**
- Увеличьте RESPONSE_DELAY
- Используйте более быструю модель
- Проверьте интернет соединение

### Резервные ответы
При ошибках API бот использует заготовленные фразы:
```javascript
const fallbackResponses = [
    'Извините, у меня проблемы с обработкой сообщения 🤖',
    'Не могу сейчас ответить, попробуйте позже 😅',
    'Что-то пошло не так с моим ИИ 🔧'
];
```
