# 📋 Полный список команд

## 🎮 Основные команды

### `!help`
Показывает список всех доступных команд с кратким описанием.

**Использование:** `!help`

**Пример:**
```
Игрок: !help
Бот: 🤖 Доступные команды:
     📍 Движение: !come, !follow <игрок>, !stop, !goto <x> <y> <z>
     ⛏️ Добыча: !mine <блок>, !dig <x> <y> <z>
     ...
```

### `!ping`
Проверка связи с ботом.

**Использование:** `!ping`

**Пример:**
```
Игрок: !ping
Бот: 🏓 Понг! Привет, Steve!
```

### `!status`
Показывает текущий статус бота: здоровье, сытость, позицию.

**Использование:** `!status`

**Пример:**
```
Игрок: !status
Бот: 📊 Статус: ❤️20/20 🍖18/20 📍(128, 64, -45)
```

## 📍 Команды движения

### `!come`
Бот подходит к игроку, который вызвал команду.

**Использование:** `!come`

**Пример:**
```
Игрок: !come
Бот: 🏃 Иду к вам, Steve!
```

### `!follow <игрок>`
Бот начинает следовать за указанным игроком.

**Параметры:**
- `<игрок>` - имя игрока для следования

**Использование:** `!follow Steve`

**Пример:**
```
Игрок: !follow Alex
Бот: ✅ Следую за Alex
```

### `!stop`
Останавливает все текущие действия бота (движение, следование, атаку).

**Использование:** `!stop`

**Пример:**
```
Игрок: !stop
Бот: ⏹️ Остановился
```

### `!goto <x> <y> <z>`
Бот идет к указанным координатам.

**Параметры:**
- `<x>` - координата X
- `<y>` - координата Y  
- `<z>` - координата Z

**Использование:** `!goto 100 64 -200`

**Пример:**
```
Игрок: !goto 0 70 0
Бот: 🎯 Иду к координатам (0, 70, 0)
```

## ⛏️ Команды добычи

### `!mine <блок>`
Бот ищет и добывает указанный тип блока поблизости.

**Параметры:**
- `<блок>` - название блока (на английском)

**Использование:** `!mine stone`

**Популярные блоки:**
- `stone` - камень
- `coal_ore` - угольная руда
- `iron_ore` - железная руда
- `diamond_ore` - алмазная руда
- `wood` - дерево
- `dirt` - земля

**Пример:**
```
Игрок: !mine coal_ore
Бот: ⛏️ Начинаю добычу coal_ore...
Бот: ✅ Добыл coal_ore!
```

### `!dig <x> <y> <z>`
Бот выкапывает блок в указанных координатах.

**Параметры:**
- `<x>` - координата X
- `<y>` - координата Y
- `<z>` - координата Z

**Использование:** `!dig 128 63 -45`

**Пример:**
```
Игрок: !dig 100 64 50
Бот: ⛏️ Копаю блок в (100, 64, 50)...
Бот: ✅ Блок выкопан!
```

## 🏗️ Команды строительства

### `!build <структура>`
Бот строит указанную структуру рядом с игроком.

**Доступные структуры:**
- `tower` - башня
- `wall` - стена
- `bridge` - мост
- `house` - дом

**Использование:** `!build tower`

**Пример:**
```
Игрок: !build house
Бот: 🏗️ Строю дом...
Бот: ✅ Дом построен!
```

### `!place <блок> <x> <y> <z>`
Бот ставит блок в указанных координатах (если есть в инвентаре).

**Параметры:**
- `<блок>` - тип блока
- `<x>` - координата X
- `<y>` - координата Y
- `<z>` - координата Z

**Использование:** `!place stone 100 65 50`

## 🎒 Команды инвентаря

### `!inventory`
Показывает содержимое инвентаря бота.

**Использование:** `!inventory`

**Пример:**
```
Игрок: !inventory
Бот: 🎒 Инвентарь:
     stone: 64
     coal: 32
     iron_ingot: 16
```

### `!craft <предмет>`
Бот крафтит указанный предмет (если есть ресурсы и рецепт).

**Параметры:**
- `<предмет>` - название предмета для крафта

**Использование:** `!craft wooden_pickaxe`

**Популярные предметы:**
- `wooden_pickaxe` - деревянная кирка
- `stone_sword` - каменный меч
- `bread` - хлеб
- `torch` - факел

**Пример:**
```
Игрок: !craft torch
Бот: 🔨 Крафчу torch...
Бот: ✅ Скрафтил torch!
```

### `!drop <предмет>`
Бот выбрасывает указанный предмет из инвентаря.

**Параметры:**
- `<предмет>` - название предмета

**Использование:** `!drop dirt`

## ⚔️ Команды боя

### `!guard <радиус>`
Бот начинает охранять территорию в указанном радиусе.

**Параметры:**
- `<радиус>` - радиус охраны в блоках (1-50)

**Использование:** `!guard 15`

**Пример:**
```
Игрок: !guard 10
Бот: 🛡️ Начинаю охрану территории в радиусе 10 блоков
```

### `!attack <цель>`
Бот атакует указанную цель.

**Параметры:**
- `<цель>` - имя игрока или моба

**Использование:** `!attack zombie`

**Пример:**
```
Игрок: !attack skeleton
Бот: ⚔️ Атакую skeleton!
```

## 💡 Советы по использованию

### Эффективное использование команд
1. **Комбинируйте команды**: Используйте `!come` перед `!follow` для быстрого подхода
2. **Проверяйте статус**: Регулярно используйте `!status` для мониторинга бота
3. **Планируйте добычу**: Используйте `!inventory` перед `!mine` для проверки места

### Автоматизация задач
```
# Последовательность для добычи:
!come              # Подойти к игроку
!mine coal_ore     # Добыть уголь
!craft torch       # Скрафтить факелы
!inventory         # Проверить результат
```

### Строительные проекты
```
# Подготовка к строительству:
!goto 100 64 100   # Идти к месту стройки
!inventory         # Проверить материалы
!build house       # Построить дом
!guard 20          # Охранять территорию
```

```
❌ Неизвестная команда: !fly
💡 Используйте !help для списка команд

❌ Не могу найти игрока Alex
💡 Проверьте правильность имени

❌ Не найдено блоков типа diamond_ore поблизости
💡 Попробуйте в другом месте или на другой глубине
```

## 🔄 Состояния бота

Бот может находиться в различных состояниях:
- **Ожидание** - готов к выполнению команд
- **Движение** - идет к цели или следует за игроком
- **Добыча** - добывает ресурсы
- **Строительство** - строит структуру
- **Охрана** - патрулирует территорию
- **Бой** - сражается с мобами

Используйте `!stop` для прерывания любого действия.
