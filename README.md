- ✅ Контекстные ответы

- ✅ Крафт предметов

### Вариант 2: Локальная установка


## ⚙️ Конфигурация

### Основные настройки (.env файл)

```env
# Сервер Minecraft
MINECRAFT_HOST=localhost
MINECRAFT_PORT=25565
MINECRAFT_VERSION=1.21.1

# Настройки бота
BOT_USERNAME=HelperBot
BOT_AUTH=offline

# OpenAI для ИИ чата
OPENAI_API_KEY=your_openai_api_key_here
OPENAI_MODEL=gpt-3.5-turbo

# Поведение бота
MAX_CHAT_HISTORY=50
RESPONSE_DELAY=1000
AUTO_RECONNECT=true
```

### Получение OpenAI API ключа

1. Зарегистрируйтесь на [OpenAI](https://platform.openai.com)
2. Перейдите в [API Keys](https://platform.openai.com/api-keys)
3. Создайте новый ключ
4. Скопируйте и вставьте в `.env` файл

## 🎮 Команды бота

### 📍 Движение
```
!come              - Подойти к вам
!follow <игрок>    - Следовать за игроком
!stop              - Остановиться
!goto <x> <y> <z>  - Идти к координатам
```

### ⛏️ Добыча ресурсов
```
!mine <блок>       - Найти и добыть блок
!dig <x> <y> <z>   - Выкопать блок в координатах
```

### 🏗️ Строительство
```
!build tower       - Построить башню
!build wall        - Построить стену
!build bridge      - Построить мост
!build house       - Построить дом
!place <блок> <x> <y> <z> - Поставить блок
```

### 🎒 Инвентарь
```
!inventory         - Показать инвентарь
!craft <предмет>   - Скрафтить предмет
!drop <предмет>    - Выбросить предмет
```

### ⚔️ Бой и защита
```
!guard <радиус>    - Охранять территорию
!attack <цель>     - Атаковать цель
```

### 🔧 Утилиты
```
!help              - Список команд
!status            - Статус бота
!time              - Время в игре
!weather           - Погода
!players           - Список игроков
!ping              - Проверка связи
```

## 🧠 ИИ Чат

Бот автоматически отвечает на сообщения игроков используя OpenAI GPT:

- **Умные ответы**: Понимает контекст игры и ситуации
- **Память**: Помнит предыдущие разговоры с каждым игроком
- **Личность**: Дружелюбный помощник с чувством юмора
- **Многоязычность**: Поддерживает русский и английский

### Примеры общения:
```
Игрок: Привет, как дела?
Бот: Привет! 👋 Отлично! Помогаю игрокам и изучаю мир. Нужна помощь?

Игрок: Где найти алмазы?
Бот: 💎 Алмазы находятся на уровне Y=11 и ниже. Попробуй !mine diamond_ore

Игрок: Построй мне дом
Бот: 🏠 Конечно! Используй команду !build house рядом с нужным местом
```

## 🐛 Решение проблем

### Бот не подключается
- ✅ Проверьте IP адрес и порт сервера
- ✅ Убедитесь что сервер онлайн и доступен
- ✅ Проверьте версию Minecraft в настройках
- ✅ Убедитесь что имя бота не занято

### ИИ не отвечает
- ✅ Добавьте валидный OpenAI API ключ
- ✅ Проверьте баланс на OpenAI аккаунте
- ✅ Упомяните бота по имени в чате
- ✅ Проверьте интернет соединение

### Команды не работают
- ✅ Убедитесь что команды начинаются с `!`
- ✅ Проверьте правильность написания команды
- ✅ Используйте `!help` для списка доступных команд

### Google Colab проблемы
- ✅ Colab бесплатно работает до 12 часов
- ✅ Используйте Colab Pro для длительной работы
- ✅ Периодически взаимодействуйте с интерфейсом
- ✅ Сохраняйте важные данные в Google Drive

## 📊 Мониторинг

Бот создает подробные логи в папке `logs/`:

- `error.log` - только ошибки

Для мониторинга в реальном времени:
```bash
tail -f logs/combined.log
```
